Adapter Design Pattern - 
I have created Adapter design pattern where Item and Cart are the Interfaces, StorePattern is concrete class, CartAdapter is adapter class implementing the Item interface and StorePattern is a concrete class implements Item.

Command Design Pattern - 
I have created Command Design Pattern where Order is the interface, OrderRequest is the request class, AcceptOrder and PendingOrder is a concrete class implements Order, OnlineStore is the broker class. In result we can see if the order is accepted or pending.

Facade Design Pattern - 
I have created Cart as interface, Vega, Loreal and Lakme as concrete classes implementing the same interface and Order as facade class.

Factory Design Pattern - 
I have createdCart as interface, Vega, Loreal and Lakme as concrete classes implementing the same interface and Order as factory class to generate object of concrete class.
